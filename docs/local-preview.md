# 本地预览

推荐使用项目根目录里的 `start-preview.cmd`。

## 使用方式

1. 打开 `WebImage` 文件夹。
2. 双击 `start-preview.cmd`。
3. 等终端显示 `URL: http://127.0.0.1:3000` 或相近端口。
4. 浏览器会自动打开本地预览页面。
5. 预览期间不要关闭终端窗口；要停止预览时按 `Ctrl+C`。

如果 3000 端口被占用，脚本会自动尝试 3001 到 3005。建议优先使用终端里显示的完整地址，不要手动改成 `localhost`。

未配置 `.env.local` 时，首页、创建页、历史页、登录页和升级页仍可预览。真实登录、扣积分、调用百炼生图和保存历史，需要先按 `docs/env.example` 配置 Supabase 与百炼环境变量。
